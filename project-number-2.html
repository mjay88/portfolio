Introduction to CSS

On this page, the plan right now is to demonstrate the styling of HTML elements. I will randomly be picking HTML elements and showing various ways we can modify there apperance in the browser through CSS. But first, what the heck is CSS?

CSS, which stands for Cascading Style Sheets, is a markup language used to dictate how websites appear visually. How far away things are from each other, the thickness of a border, the shadow of some header text, background colors, the font type; are all controlled with CSS. CSS is pretty straight forward from what I understand, the hardest part being appreciating the size and scope of what is possible with CSS, the type of appreciation that I know will only come with months and months of practice.

CSS can be applied three ways. We can include CSS directly inline on each element, but this is typically not considered best practice. CSS can be included in the head tag of an HTML document inside of a style tag as well. This method is not considered good practice either. Using these methods will drastically limit the true potential of what CSS can do, and your mark-up will be redundant and unnecesarily verbose. The best excepted practice is the external stylesheet way. This method, you create a completely different type of file, a .css file, and link it to your html files through specfications in the html files head tag.

Break it down

Ruleset Desciption
Selector This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example, paragraph elements). To style a different element, change the selector.
Declation This is a single rule like color: red;. It specifies which of the element's properties you want to style.
Properties These are ways in which you can style an HTML element. (In this example, color is a property of the elements.) In CSS, you choose which properties you want to affect in the rule.
Property Value To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.)
